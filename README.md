# Miniguia de Estudo

    Protocolo Definição: Regras que padronizam a comunicação entre dispositivos
    . Exemplo de uso: Permitir que computadores de marcas diferentes troquem dados

    Modelo OSI Definição: Modelo de 7 camadas para padronizar a interconexão de sistemas
    . Exemplo de uso: Isolar se um erro é físico (cabos) ou lógico (aplicativo)
        
    Switch Definição: Dispositivo que envia dados apenas ao destino correto via endereço físico
    . Exemplo de uso: Reduzir o tráfego desnecessário em redes locais
        
    Roteador Definição: Equipamento que interliga redes e escolhe a melhor rota para os dados
    . Exemplo de uso: Conectar uma rede doméstica à Internet
    
    Endereço IP Definição: Identificação lógica de um aparelho em uma rede
